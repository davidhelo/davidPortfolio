AI tic-tac-toe game

This project is a web application to play tic-tac-toe against the machine.
It implements Minimax algorithm in an API to calculate the optimal move.

Live Link

Project Overview

Minimax is a type of Adversarial Search algorithm used in AI for games.

This app is an implementation in Python of the minimax algorithm. A correct implementation results in an unbeatable machine, you won't be able to win against it (optimal play in both sides always results in a tie).
This algorithm recursively searches all possible paths up to the last outcome (winner or tie) and assign a score to each one that is compare to other paths scores to maximize the possibility to win.

The web app's frontend was build using React JS and the backend using Django Framework. As the React App is pre-build, you'll find the source code for the React App in a different repository.

Find more information about Adversarial Search and Minimax algorithms in this lecture from Harvard's course cs50AI , or more information from geeks for geeks here.
